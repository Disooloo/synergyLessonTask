Терминология:

Выражение (expression) - комбинация переменных, операций и констант, которую можно вычислить.
    Операция (operation) - действие, которое выполняется над одним или несколькими операндами для получения результата.
    Переменная (variable) - обозначение для хранения значения, которое может меняться в процессе выполнения программы.
    Операнд (operand) - значение, над которым выполняется операция.
    Тип операции - группировка операций по их функциональности, например, арифметические, логические, присваивания и т.д.
    Бинарные операции:

а. Арифметические операции:

Сложение (+) - складывает два операнда.
    Вычитание (-) - вычитает из одного операнда другой операнд.
    Умножение (*) - умножает два операнда.
    Деление (/) - делит один операнд на другой операнд.
    Деление по модулю (%) - возвращает остаток от деления одного операнда на другой операнд.
    б. Операции присваивания:

Присваивание (=) - присваивает значение одного операнда другому.
    в. Логические операции:

Логическое И (&&) - возвращает true, если оба операнда истинны.
    Логическое ИЛИ (||) - возвращает true, если хотя бы один операнд истинный.
    Логическое НЕ (!) - инвертирует значение операнда.
    г. Операции сдвига:

Сдвиг влево (<<) - сдвигает биты операнда влево на заданное количество позиций.
    Сдвиг вправо (>>) - сдвигает биты операнда вправо на заданное количество позиций.
    д. Преобразования типов:

Преобразование целочисленного типа в вещественный тип (float) - производится при помощи операции присваивания, когда целочисленный тип присваивается переменной типа float.
    Преобразование вещественного типа в целочисленный тип (int) - производится при помощи операции присваивания, когда вещественный тип присваивается переменной типа int.
    Приоритеты операций:

При выполнении выражения важно учитывать приоритет операций, который определяет порядок выполнения операций. Обычно приоритет операций определяется стандартной алгебраической нотацией, где операции с наивысшим приоритетом выполняются
